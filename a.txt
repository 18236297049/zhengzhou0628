git config --global user.name：查看用户名，加用户名为修改
git config --global user.email：查看邮箱，加邮箱为修改
ssh-keygen -t rsa -C 邮箱：获取密钥
git remote：查看远程仓库

git commit 
feat：新功能（feature）
fix：修补bug
docs：文档（documentation）
style： 格式（不影响代码运行的变动）
refactor：重构（即不是新增功能，也不是修改bug的代码变动）
test：增加测试
chore：构建过程或辅助工具的变动
igin

git push origin（远程仓库名） main（分支名）
git reflog ：查看版本号
git reset hard 版本号 回退版本

git checkout -b 分支名：创建分支
git checkout  分支名：切换分支
git branch：查看本地分支
git branch -a：查看远程分支
